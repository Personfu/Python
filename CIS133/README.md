# Phase 02 — Software Engineering
**Department:** Engineering | **Course:** CIS 133 — Intro to Programming | **Status:** Complete

---

## Purpose

Build FLLC's core software engineering capability — 8 production Python modules covering the full spectrum from basic I/O to object-oriented design patterns. Every module follows FLLC coding standards with comprehensive documentation.

## Deliverables

| # | File | Description | Key Concepts |
|---|------|------------|-------------|
| 1 | [`1_hello_world_basic_math.py`](1_hello_world_basic_math.py) | User input and salary calculator | `input()`, `float()`, arithmetic, `format()` |
| 2 | [`2_variables_and_types.py`](2_variables_and_types.py) | Temperature conversion tool | Variable types, conversion formulas, formatted output |
| 3 | [`3_strings_and_things.py`](3_strings_and_things.py) | String manipulation suite | `len()`, slicing, concatenation, `join()`, lists |
| 4 | [`4_movie_age_check.py`](4_movie_age_check.py) | Age-based pricing engine | `if/elif/else` branching, comparison operators |
| 5 | [`5_loops_and_iteration.py`](5_loops_and_iteration.py) | Iteration patterns and algorithms | `for`, `while`, `break/continue`, `enumerate`, nested loops, list comprehensions, patterns (pyramid/diamond), Fibonacci, primes, FizzBuzz |
| 6 | [`6_functions.py`](6_functions.py) | Function design patterns | Parameters, `*args/**kwargs`, lambda, recursion (factorial/Fibonacci/binary search), scope, type hints, decorators |
| 7 | [`7_file_io.py`](7_file_io.py) | File processing framework | Text/CSV/JSON read/write, `os.path`, error handling, log analyzer |
| 8 | [`8_classes_oop.py`](8_classes_oop.py) | Object-oriented design | Constructors, properties, inheritance, polymorphism, abstract classes, composition, magic methods |

## Enterprise Application

- **Phase 05** extends OOP patterns to build data structure libraries (linked lists, BST, graphs)
- **Phase 07** uses Python for encryption toolkit (hashlib, hmac, secrets)
- **Phase 09** applies all patterns in the production REST API server
