# Capstone Project Proposal
## CIS 425 — Capstone | Preston Furulie

## Project Title
**Guitar Shop Enterprise Platform — Full-Stack Cloud Deployment**

## Problem Statement
Small-to-medium music retailers lack affordable, integrated solutions for managing inventory, processing orders, and analyzing sales data across multiple locations.

## Proposed Solution
A full-stack web application with:
- Real-time inventory management across warehouses
- Customer-facing e-commerce storefront
- Admin dashboard with analytics and reporting
- RESTful API serving mobile and web clients

## Technology Stack

| Layer          | Technology                        |
|----------------|----------------------------------|
| Frontend       | React 18, TailwindCSS            |
| Backend        | Node.js, Express                 |
| Database       | MySQL 8.0, Redis (cache)         |
| Infrastructure | AWS (ECS, RDS, ALB, CloudFront)  |
| CI/CD          | GitHub Actions, Docker           |
| Monitoring     | CloudWatch, custom health checks |

## Timeline

- **Weeks 1-2:** Requirements gathering, database design
- **Weeks 3-5:** Backend API development and testing
- **Weeks 6-8:** Frontend development and integration
- **Weeks 9-10:** Cloud deployment, CI/CD pipeline
- **Week 11:** Load testing and security audit
- **Week 12:** Final presentation and documentation
