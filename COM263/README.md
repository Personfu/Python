## Course Overview

COM 263 is a supporting course that develops interpersonal communication skills
essential for IT professionals. Effective communication is a critical competency
in technology teams, where collaboration, documentation, and stakeholder management
directly impact project success.

This course covers communication theory, technical writing, and presentation skills
with a focus on applying these concepts in professional IT environments.

## Learning Objectives

- Apply communication models (Linear, Interactive, Transactional) to workplace scenarios
- Demonstrate active listening and empathetic communication techniques
- Analyze cultural differences in communication (high-context vs low-context)
- Apply conflict management styles appropriate to professional situations
- Write clear technical documentation (SOPs, runbooks, incident reports)
- Design and deliver effective technical presentations
- Communicate effectively with both technical and non-technical stakeholders
- Apply the Johari Window model for self-awareness and team building
